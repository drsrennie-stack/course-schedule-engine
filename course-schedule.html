How to import each file

Calendar (.ics): In Google Calendar, open Settings, then Import and export, then Import. In Apple Calendar, choose File, then Import.

Anki deck (.csv): In Anki, choose File, then Import, pick the file, set the field separator to Comma, and map Front, Back, Deck, and Tags. The file already names a deck per week.

Study plan (.txt): Open it in any text editor or paste it into your planner. Each date lists the recommended tasks and which topics to review that night.

Planner checklist (.md): Paste directly into Notion, Obsidian, or Apple Notes. For Todoist or Things, paste into the quick-add box, each line becomes a task.

SLO grade map (.csv): Import into your grade or SLO tool as a CSV. The first row is the header. One row per graded item, with the mapped SLO codes.
